Costcenter ¶

Costcenters can be used to categorize entries. Entries can be linked to a costcenter. Costcenters are children of a period.

Costcenter List  ¶

Get Costcenter List
GET/costcenter{?filter,order,format}

Lists all available costcenter IDs.

Example URI

GET /costcenter?filter=&order=title ASC&format=
URI Parameters
HideShow
filter
string (optional) 

Filter the list using the Query Language

order
string (optional) Example: title ASC

Sort the costcenter list by property and direction

format
string (optional) 

Specify format=full to get the full object instead of a list of IDs

Response  200
HideShow
Headers
Content-Type: application/json
Body
{
  "objects": [
    1098,
    1864
  ]
}
